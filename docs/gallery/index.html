This is the third year this unit is taught at the University of Bristol. During the first two years there has been some amazing work produced as extensions to the labs. Below is a gallery of some of this work.

Hal Jones & Bhavish Jogeeah
Very Detailed raytracer that contains pretty much everything that we did in the unit. The most impressive thing with this work is that the techniques used are not just straight forward implementations but actually done in quite different ways. This is a great example of thinking outside the box. Hal will be teaching on this unit this year so make sure that you have a chat with him about how this images was created.

link

Qingqi Shi & Weijia Liu
Beautiful composition in this image. Graphics is not just about throwing a load of math at an image but actually understanding what makes a visual difference and what does not. In this project it has been achieved to perfection.

link

Michael Fitzpatric & Michael Parkin-White
This raytracer contains a lot of different techniques, the most important thing is the photon mapper and final gether which generates very stable results as can be seen below the glas dragon. Anti-Aliasing, Textures, Area Lights/Soft Shadows, Object Loader, Materials and scene management, Photon Mapping and Final Gather, Octree, Caustics, Image filters

link

Michael Fitzpatric & Michael Parkin-White
It is not often you get goose-bumps marking coursework, this project by Michael and Michael gave me just this. I felt gulity giving this a mark of 100%! Click on the link below and be prepared to be amazed.

link

Jamie Willis & Harry Mumford-Turner
This raytracer contains a lot of techniques. It uses photon mapping for global illumination, soft-shadows, anti-aliasing etc. To make it all run in a sensible time lots of optimisations have been implemented, it uses KD-Trees for the geometry. The colour bleeding and other effects are really subtle and makes for a very balanced image, really like this one. Jamie will be teaching on the unit this year so make sure that you have a chat with him about his work.

link

Jamie Willis & Harry Mumford-Turner
This is a really speedy rasteriser where a lot of care have been taken to make sure that the memory access is done correctly. Further, the implementation of softshadows, is really high quality. Have a look at the video linked below to see it in full flow.

link

Rory Smith & Julian Loscombe
Very good raytracer including a lot of optimisations, it uses KD-trees to store the geometry, OpenCL and OpenMP to optimise computations. It also implements, glass, global illumination and mirrors among other things.

link

Rory Smith & Julian Loscombe
Excellent rasteriser engine, it does full clipping, has texture mapping and normal maps, back-face culling for speed and also uses an interesting technique to optimise the triangle filler.

link

John Griffith & Matthew Duffin
Lovely rasterition engine which includes, object loader, shadows, material and full clipping. It is also very fast due to some clever optimisations and well crafted code.
